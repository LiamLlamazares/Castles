# Sound Files Required

This folder should contain the following sound files for game audio feedback:

## Required Files
- `move.mp3` - Piece movement sound
- `capture.mp3` - Piece capture sound  
- `castle.mp3` - Castle capture sound
- `victory.mp3` - Game won sound
- `defeat.mp3` - Game lost sound

## Recommended Sources

### Lichess Sounds (MIT Licensed)
Download from: https://github.com/lichess-org/lila/tree/master/public/sound

### Free Chess Sound Effects
- freesound.org (search "chess move")
- opengameart.org

## Volume Guidelines
- Keep sounds short (< 0.5 sec for moves)
- Normalize volume levels between files
- Use MP3 format for browser compatibility
